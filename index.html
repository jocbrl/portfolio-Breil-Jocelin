SYSTEM ADMINISTRATION

In charge of modernizing the complete information system of a company to ensure adherence to standards and seamless integration with another company.
In the project, we upgraded and secured the company's information system during the acquisition process. This involved implementing an Active Directory, firewall, VPNs, and containerization. We also established a backup plan to mitigate risks and ensure business continuity. These measures ensured compliance with security standards and enhanced system performance.

SYSTEM AND NETWORK

During the project, we were tasked with configuring the computer network for specific locations within a city: eXia, the library, the ENGIE building, the DIGIPLEX building, and the DATACENTER. Each building had specific requirements to fulfill. For instance, the eXia building needed a DHCP, DNS, and FTP server. The ENGIE and DIGIPLEX buildings required different VLANs, inter-VLAN routing, and SSH configuration across all switches. Additionally, the DATACENTER had to be configured for IPv6, with an IPv6 tunnel established between it and the eXia building.

EMBEDDED SYSTEM

In this project, we created a miniature weather station prototype using electronic components. It collected data and stored it on an SD card for analysis. The weather station featured multiple modes: Standard mode (green LED): Records data onto the SD card every 10 minutes. Configuration mode (yellow LED): Allows for system settings modifications. Maintenance mode (orange LED): Data is accessible but not written to the SD card; the card can be safely removed. Economy mode (blue LED): Reduces data acquisition to save power. The weather station included sensors, LEDs, and buttons. Its pre-set parameters controlled measurement intervals and clock configuration for accurate dating of data.
